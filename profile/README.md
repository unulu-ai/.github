**What AI agents can do with unulu:**

- Create link-in-bio sites with name, bio, links, and 17 built-in themes
- Read and update existing sites programmatically
- Check handle availability on unu.lu
- Publish instantly — no authentication required

**MCP server:** `https://mcp.unulu.ai` (streamable HTTP, no auth)
**REST API:** `https://api.unulu.ai` ([OpenAPI spec](https://unulu.ai/openapi.json))

Starting with link-in-bio sites. Portfolios, shops, and more on the way.
